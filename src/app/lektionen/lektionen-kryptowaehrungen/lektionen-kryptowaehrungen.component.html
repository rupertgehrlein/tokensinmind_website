Kryptowährungen

Noch mehr Krypto?
Das Wort Kryptowährung kommt tatsächlich von der Kryptographie, denn hier geht spielt das Geheimhalten von Informationen eine riesen Rolle. Kryptowährungen gibt es viele, aber Eins haben sie alle gemeinsam. Kryptowährungen können nur im Computernetz verwendet werden, weil sie digitale Währungen sind. Man kann genauso wie mit Euro oder Dollar damit bezahlen, aber in der Hand halten kann man sie nicht.

Eine weitere Besonderheit der Kryptowährung ist, dass sie nur von der Öffentlichkeit abhängen, sie werden also von keinem Staat überprüft. Auch kann mit Kryptowährungen Geld verschickt werden, ohne dass man eine Überweisung über die Bank macht. Die Idee: eine freie und digitale Währung, die gleichzeitig auch komplett anonym ist.

Wie kann das was Wert sein?
Einen Wert für Kryptowährungen festzulegen ist tatsächlich sehr schwierig, weil sie rein digital sind. Darum muss das anders funktionieren, als beispielsweise bei Gold. Der Preis wird bei Kryptowährungen nur von den Kunden bestimmt. Jeder, der kauft oder verkauft, entscheidet wieviel er selbst bezahlen oder erhalten möchte.
