Kryptowährungen

Noch mehr Krypto?
Das Wort Kryptowährung kommt tatsächlich von der Kryptographie, denn hier geht spielt das Geheimhalten von Informationen eine riesen Rolle. Kryptowährungen gibt es viele, aber Eins haben sie alle gemeinsam. Kryptowährungen können nur im Computernetz verwendet werden, weil sie digitale Währungen sind. Man kann genauso wie mit Euro oder Dollar damit bezahlen, aber in der Hand halten kann man sie nicht.

Eine weitere Besonderheit der Kryptowährung ist, dass sie nur von der Öffentlichkeit abhängen, sie werden also von keinem Staat überprüft. Auch kann mit Kryptowährungen Geld verschickt werden, ohne dass man eine Überweisung über die Bank macht. Die Idee: eine freie und digitale Währung, die gleichzeitig auch komplett anonym ist.

Wie funktionieren Kryptowährungen?
Kryptowährungen nutzen Blockchains. In einer Blockchain können sehr viele Informationen gespeichert werden. Das kann bei Kryptowährungen genutzt werden, um Transaktionen aufzuschreiben, also jeder Kauf/Verkauf wird in einen Block geschrieben. Durch einen Mechanismus wird sichergestellt, dass die Information auf jedem Computer auf dem gleichen Stand sind. So können die Infos nicht verändert werden. Außerdem sind die Daten dezentral gespeichert, also nicht nur an einem Ort, sondern auf ganz vielen Computern.

Wie kann das was Wert sein?
Einen Wert für Kryptowährungen festzulegen ist tatsächlich sehr schwierig, weil es sich um Etwas rein digitales handelt. Bei Gold z.B. ist das sehr einfach, weil das Material einen bestimmten Wert hat. Wie macht man das aber bei etwas, das es nichteinmal in Echt gibt? Tatsächlich wird der Preis bei Kryptowährungen allein von den Nutzern bestimmt. Jeder, der kauft oder verkauft, entscheidet wieviel er selbst bezahlen oder bekommen möchte. Der Wert der Währung hängt also nur vom Markt ab.
