Blockchain

Was ist eine Blockchain?
Eine Blockchain ist eine Sammlung von riesigen Mengen an Daten. Man kann dabei auch von einer Art Datenbank sprechen. Die Daten werden in einen einzelnen Block gepackt und es wird dazugeschrieben an welchen anderen Block die Daten angekettet werden sollen. Unser Block wird dann sicher verschlossen und er wird in ein Netzwerk aus Computern gespeichert. So entsteht eine Kette von Blöcken - die Blockchain.

Was ist so besonders an der Blockchain?
Die Informationen in einer Blockchain bleibt für die Ewigkeit. Ist ein Block einmal an der Kette angehängt, kann er nicht mehr so leicht geändert werden. Das liegt Erstens an der Hashfunktion, die die Daten im Block unlesbar und vor allem einzigartig macht. Zweitens sind die Informationen an ganz vielen verschiedenen Stellen gespeichert. Es müsste der Block überall geändert werden, wo es eine Kopie von ihm gibt. So kann ein Block nicht verändert werden, wenn er einmal im Netzwerk ist.

Wofür wird eine Blockchain verwendet?
Blockchains werden in vielen verschiedenen Bereichen eingesetzt, zum Beispiel:
  • In der Finanzwelt werden Blockchains verwendet, um Kryptowährungen wie Bitcoin zu speichern.
  • In der Logistik werden Blockchains verwendet, um Lieferketten zu verfolgen.
  • In der Gesundheitsversorgung werden Blockchains verwendet, um Patientendaten zu speichern.
Ein Beispiel
Jedes Mal, wenn ihr euch trefft, schreibt ihr auf, wer dabei war und was gespielt wurde. Diese Liste könnt ihr dann an alle Freunde schicken, damit jeder weiß, was passiert ist. Wenn jemand etwas ändern möchte, muss er alle anderen Freunde darüber informieren. Wenn jemand versucht, eine Liste zu ändern, ohne alle anderen Freunde darüber zu informieren, wird das sofort bemerkt. So ähnlich funktioniert eine Blockchain auch.
Wie ist ein einzelner Block in der Blockchain aufgebaut?
Ein einzelner Block in der Blockchain ist wie ein kleines Paket, das Informationen enthält. Dieses Paket wird von einem Computer erstellt und dann an alle anderen Computer im Blockchain-Netzwerk verteilt.

Was ist in einem Block enthalten?
In einem Block sind folgende Informationen enthalten:
  • Der Hashwert des vorherigen Blocks. Dieser Hashwert ist eine Art Fingerabdruck, der den vorherigen Block ausmacht.
  • Die Transaktionen. Dies sind die Informationen, die in diesem Block gespeichert werden.
  • Der Zeitstempel. Dieser Zeitstempel zeigt an, wann der Block erstellt wurde.
  • Der Hash des Blocks. Dieser Hash ist eine Art Fingerabdruck, der den aktuellen Block identifiziert.
Wie werden die Informationen in einem Block gesichert?
Die Informationen in einem Block werden durch einen Hash gesichert. Ein Hash ist eine mathematische Funktion, die eine Eingabe in eine feste Zeichenkette ausgibt. Wenn man eine Information in einen Hash umwandelt, ist es sehr schwierig, diese Information wiederherzustellen. Das bedeutet, dass die Informationen im Block nicht geändert werden können, ohne den Hash zu ändern.

Wie werden die Blöcke miteinander verbunden?
Die Blöcke in einer Blockchain sind miteinander verbunden, indem der Hash des vorherigen Blocks in den aktuellen Block geschrieben wird. Diese Verbindung sorgt dafür, dass die Blöcke in einer bestimmten Reihenfolge angeordnet sind.

Was ist die NONCE in einem Block?
Die NONCE ist eine Abkürzung für "number used once", also eine "Zahl, die nur einmal verwendet wird". Jeder Block einer Blockchain hat eine NONCE. Sie wird von dem Computer generiert, der einen neuen Block erstellen möchte.

Wofür gibt es die NONCE?
Die NONCE wird verwendet, um den Hash des Blocks zu generieren. Man verändert sie also so lange, bis der Hash so aussieht, wie er eben aussehen soll. Der Hash an sich ist meistens schon sicher genug, aber manchmal muss man wichtige Informationen doch noch ein bisschen besser schützen.

Wie funktioniert die NONCE?
Die NONCE wird zusammen mit den anderen Informationen in einem Block verwendet, um einen Hash zu generieren. Um die Sicherheit zu erhöhen kann man vorbestimmen wie der Hash aussehen soll, z.B. soll der Hash mit 4 Nullen am Anfang starten. Der Computer, der den neuen Block erstellen möchte, muss also die NONCE so lange ändern, bis der Hash die gewünschte Anzahl von Nullen hat. Dadurch wird das knacken des Hashs noch schwieriger. Das ganze kannst du in den Übungen mal ausprobieren.

Wieso ist die Blockchain so sicher?
Die Blockchain verbindet verschiedene informatische Prinzipien, damit die Daten in ihr sicher und unveränderlich bleiben. Du hast bereits bei den Lektionen über Kryptographie etwas über das Hashing gelernt. Vielleicht hast du auch schon bei den Übungen das Hashing selbst einmal ausprobiert. Dann hast du bestimmt gemerkt, dass die kleinste Änderung auf einmal einen ganz anderen Hashwert ergibt. Da alle Blöcke miteinander verbunden sind über eben diesen Hashwert kann sofort bemerkt werden, wenn jemand versucht hat Daten zu ändern. Auch das kannst du einmal bei unseren Übungen ausprobieren.

Aber was hindert denn jemanden daran trotzdem etwas in einem Block zu ändern?
Hier kommt die nächste Stufe der Sicherheit bei der Blockchain ins Spiel. Wenn das schwierige mathematische Rätsel von einem Computer in der Blockchain gelöst wird und dadurch ein neuer Block entsteht, schauen alle anderen Computer in dem Computer-Netzwerk erst einmal ob auch alles richtig ist. Erst wenn geprüft wurde, dass alles richtig ist, wird der neue Block mit den anderen verbunden. Genau das verhindert nun auch, dass etwas falsches errechnet werden kann oder das ein Computer absichtlich falsche Daten in die Blockchain schreibt. Schließlich muss erst einmal die Mehrheit aller Computer davon überzeugt werden, dass das was in die Blockchain eingefügt wird auch richtig ist. Die allermeisten Teilnehmer bei einer Blockchain wollen aber natürlich, dass da nichts falsches steht. Deswegen ist das Ganze so sicher. Wenn nun also jemand etwas böses machen möchte, muss er die Mehrheit davon überzeugen mitzumachen. Das ist zwar theoretisch möglich, in der echten Welt aber sehr sehr unwahrscheinlich.

Ist das nicht schädlich für unsere Umwelt?
Hier streiten sich die Menschen und vergleichen die Vor- und die Nachteile. Insgesamt lässt sich aber sagen, dass viele Varianten der Blockchain-Technologie sehr viel Strom verbrauchen. Hier gibt es verschiedene Arten und Weisen wie eine Blockchain arbeitet und dafür sorgt, dass alle Daten in ihr auch richtig sind. Die häufigste Art heißt "Proof-Of-Work". Das ist englisch und heißt übersetzt sowas wie "Arbeitsnachweis". Hier rechnen ganz viele Computer um die Wette und am Ende hat nur einer von ihnen das richtige Ergebnis gefunden und erhält dafür eine Belohnung. Das kostet natürlich ganz viel Energie. Nicht nur weil der Computer der die Lösung gefunden hat so viel arbeiten musste, sondern auch weil alle anderen Computer, die die Lösung nicht gefunden haben, quasi umsonst gearbeitet haben. Gerade wenn der Strom nicht aus erneuerbaren Energien wie Windkraft oder Solaranlagen stammt, dann ist das natürlich ein großes Problem. Zusätzlich entsteht hierbei deutlich mehr Elektromüll, als wenn es die Technologie der Blockchain nicht geben würde. Wenn ein Computer ständig schwer arbeitet, dann geht auch häufiger mal ein Teil kaputt. Das Teil wird dann weggeworfen und durch ein Neues ersetzt. Gerade durch die harte Arbeit dieser Computer in einem Blockchain-Netzwerk passiert das deutlich öfter als bei unseren Computern daheim.
